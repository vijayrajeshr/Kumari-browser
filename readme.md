# Simple WebView Browser with SaltUI and EinkBro

## Project Overview
This project aims to create a **simple, highly functional web browser** with enhanced reading features. It integrates two open-source libraries:
- **SaltUI** (from https://github.com/Moriafly/SaltUI) for a beautiful, modern UI.
- **EinkBro browser** (from https://github.com/plateaukao/einkbro) for advanced reading and webview functionality.

The goal is a fast, responsive Android or Flutter WebView browser with rich reading features and a visually appealing interface.

## Key Features
- Modern UI using SaltUI components.
- WebView-based browser capabilities from EinkBro.
- Reading mode enhancements (e.g., text adjustment, dark mode).
- Fast loading and smooth user interaction.
- Easily extendable with new reading and browsing features.

## Technology Stack
- Android Studio (Kotlin) or Flutter (your preference).
- SaltUI Compose Multiplatform for UI.
- EinkBro as the base browser engine on Android WebView.

## Getting Started
1. Clone or fork this repo.
2. Add SaltUI and EinkBro dependencies.
3. Integrate SaltUI for the UI layer.
4. Embed EinkBro WebView component for browsing.
5. Implement reading features such as:
   - Text resizing.
   - Night mode.
   - Article simplification.
6. Test on Android devices or Flutter environments.

## Objectives
- Quickly build a functional browser app.
- Deliver a smooth reading experience.
- Maintain clean, reusable UI code with SaltUI.
- Leverage EinkBro's browser engine for reliable performance.
